Mantis Blades

Completion: May 1, 2025

3D Printed & Working Recreation of Arasaka Mantis Blades from Cyberpunk 2077 This project is intended to be used for cosplays of the game as well as being a semi-functional mechanical extension of our own arms. The system utilizes a few electronic components that help motorize its different parts, allowing smooth automated movement. Recommended to print in PLA/PETG, can be done in a few days.

TrimTab

Completion: October 2, 2024

A 3D color digital rendering of Dr. Eugene Tssui's proposed eco-friendly coastal housing. This project is an organic shark-dorsal-fin shaped building for tidal wave proof residence. Designed and modeled using Rhino 3D with help from Dr. Eugene Tssui and Cal Poly students. Able to cut through a tidal wave coming directly towards the building, designed for low power emissions.
